# Raidon Homebrew Tap

Homebrew formulae for Raidon tools.

## Installation

```bash
brew tap raidon-inc/tap
brew install p3
```

Or in one command:

```bash
brew install raidon-inc/tap/p3
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| `p3` | [TaskP3 CLI](https://www.taskp3.com/cli) - manage tasks from the command line |

## Updating

```bash
brew update
brew upgrade p3
```
